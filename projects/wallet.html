MultiCurrency HD Crypto Wallet

A web-based hierarchical deterministic (HD) wallet supporting multiple cryptocurrencies. Create/restore wallets, manage accounts, view balances, and send signed transactions via blockchain node APIs.

Project Overview

  • Type: Web Application
  • Tech: HTML, JavaScript
  • APIs: Blockchain shared nodes (e.g., NowNodes)
  • Coins: Ethereum, Tether (ERC-20)
  • Year: 2023

Abstract

This project implements a multi-currency HD wallet where users can create or restore a wallet using a seed phrase, generate accounts/addresses deterministically, fetch balances from blockchain node APIs, and send signed transactions.

What I Did

  • Implemented the wallet UI and core logic in JavaScript
  • Built create/restore flow using seed phrases and deterministic key derivation
  • Handled account/address generation and basic wallet state management
  • Integrated blockchain APIs to fetch balances and broadcast transactions
  • Implemented transaction signing and sending flow

Technical Highlights

  • Hierarchical Deterministic (HD) wallet concepts
  • Public/private key generation and management
  • Transaction signing and broadcasting
  • REST API integration with blockchain nodes
  • Frontend state handling in JavaScript
