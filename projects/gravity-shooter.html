Abstract

Gravity Shooter is an innovative 2D multiplayer game that blends fast-paced gameplay with interactive learning in physics. Players navigate dynamic environments filled with obstacles while using gravity-based mechanics to aim, shoot, and solve challenges, turning core physics concepts like trajectory, force, and gravity into hands-on gameplay.

The game includes a machine learning–powered training mode that analyzes the player’s position and target distance in real time and provides smart aiming suggestions. Players can then jump into solo or multiplayer modes, where mechanics like manipulating opponents’ gravity add a strategic twist.

  • Built the ML-based training mode using Unity ML-Agents (PPO) for real-time aiming assistance

Key Features
