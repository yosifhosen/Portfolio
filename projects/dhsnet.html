DHSNet – Salient Object Detection

A computer vision project focused on detecting the most visually important objects in an image using a deep hierarchical neural network architecture with coarse-to-fine refinement.

Project Overview

  • Field: Machine Learning / Computer Vision
  • Task: Salient object detection (saliency maps)
  • Architecture: GV-CNN + HRCNN (VGG-based)
  • Datasets: MSRA10K, DUT-OMRON, ECSSD, PASCAL-S
  • Output: Saliency maps highlighting important objects
  • Year: 2023

What I Did

  • Studied and implemented the DHSNet architecture for saliency detection
  • Worked with VGG-based CNNs and hierarchical recurrent CNNs (HRCNN)
  • Built the training and validation pipeline using multiple benchmark datasets
  • Handled preprocessing, resizing, and dataset splits
  • Evaluated results against ground truth saliency masks and compared outputs

Technical Highlights

  • Convolutional Neural Networks (VGG-16)
  • Hierarchical Recurrent CNNs (HRCNN)
  • Multi-stage coarse-to-fine prediction pipeline
  • Training with backpropagation on GPU
  • Dataset-driven evaluation and benchmarking
